# Archived Course Files

Archived on 2026-05-16 while separating personal course material from the MyWeb application.

Contents:

- `courses/`: former QuestionGen course registry, courseware, simulations, review notes, and fixtures.
- `questiongen-data/`: older OCR/course material that previously lived under `backend/questiongen-data`.
- `docs-course-materials/`: stray course documents and generation scripts that previously lived under `docs/`, including Java 2025 notes and SEDP quiz/mock exam files.

These files are kept for reference only. MyWeb no longer treats them as active application data. To restore QuestionGen courseware locally, move or copy `archive/course-files/courses/` back to the repository root as `courses/`.

## 🔍 购买服务器时的注意事项

1. **操作系统选择**: 
   - 推荐 Ubuntu 20.04 LTS 或 22.04 LTS
   - 避免使用 CentOS（已停止维护）

2. **地域选择**:
   - 国内用户：选择离目标用户最近的地域（如华东、华北）
   - 海外用户：选择美国或新加坡

3. **网络配置**:
   - 确保分配公网 IP
   - 开放 80 和 443 端口
   - 带宽至少 1Mbps

4. **安全组/防火墙**:
   - 开放端口：22 (SSH), 80 (HTTP), 443 (HTTPS)
   - 关闭不必要的端口

5. **付费方式**:
   - 按量付费：灵活，适合测试
   - 包年包月：优惠，适合长期使用

## 📝 购买后的第一步

1. **记录信息**:
   ```
   服务器公网IP: ______________
   SSH端口: 22
   用户名: root
   密码: ______________
   ```

2. **首次连接**:
   ```bash
   ssh root@your_server_ip
   ```

3. **基础配置**:
   ```bash
   # 更新系统
   apt update && apt upgrade -y
   
   # 修改密码（重要！）
   passwd
   
   # 创建非root用户
   adduser kitchenbook
   usermod -aG sudo kitchenbook
   ```

4. **开始部署**:
   按照 [DEPLOYMENT.md](DEPLOYMENT.md) 或 [quick-start.md](quick-start.md) 继续

## ❓ 常见问题

**Q: 我需要备案吗？**
A: 如果使用国内服务器（阿里云、腾讯云等）且绑定域名，需要ICP备案。使用IP访问或海外服务器不需要。

**Q: 备案要多久？**
A: 首次备案通常需要15-20个工作日。

**Q: 没有域名可以部署吗？**
A: 可以，直接使用服务器IP访问即可。但不能使用HTTPS（除非自签名证书）。

**Q: 忘记服务器密码怎么办？**
A: 登录云服务商控制台，可以重置密码或通过VNC直接连接。

**Q: 如何选择操作系统版本？**
A: 推荐 Ubuntu 22.04 LTS，稳定且软件源更新。避免使用非LTS版本。
